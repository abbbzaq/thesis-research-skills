# Research Skills for Codex

This repository contains four practical skills for graduate study and research workflows:

- `paper-reading-matrix`
- `research-plan-refiner`
- `thesis-writing-assistant`
- `experiment-log-organizer`

These skills are designed for frequent academic use cases such as:

- reading and comparing papers
- refining research ideas into executable plans
- improving thesis and paper writing
- organizing experiment logs for reports and ablations

Each skill includes:

- `SKILL.md`
- `agents/openai.yaml`
- `assets/`
- `references/`
- `scripts/`
